# الرموز والحقول (حوار البنية)

تدير نافذة **الرموز والحقول** تعريفات بنية المشروع: **رموز الأنشطة** (أبعاد قابلة للتعريف بحرية مثل الموقع أو التخصص) و**الحقول المخصصة** (حقول مستخدم بأنواع محددة). تُملأ القيم بعد ذلك لكل مهمة عبر لوحة الخصائص أو [حوار المهمة](docs://ref-taakdialoog).

## الفتح

**التخطيط** ← مجموعة الشريط **الهيكل** ← **الرموز والحقول**. **Esc** أو علامة الإغلاق أو نقرة خارج النافذة يغلقها. تسري جميع التغييرات فورًا (ويمكن التراجع عنها بـ Ctrl+Z) — لا يوجد زر حفظ منفصل.

## رموز الأنشطة

"أبعاد قابلة للتعريف بحرية (مثل الموقع، التخصص) للتجميع والتصفية — قيمة واحدة كحد أقصى لكل نوع لكل مهمة."

كتلة واحدة لكل نوع رمز:

- **اسم نوع الرمز** — قابل للتعديل مباشرة.
- **إزالة نوع الرمز** (سلة مهملات) — يزيل النوع بما فيه كل القيم والتخصيصات على المهام.
- صف واحد لكل قيمة: **الرمز** (تسمية قصيرة)، **الوصف** ومنتقي **اللون** (يلوّن التجميعات ضمن أمور أخرى)، إضافة إلى زر إزالة.
- **إضافة قيمة** — قيمة جديدة تحت هذا النوع.

في الأسفل: حقل إدخال **نوع رمز جديد (مثل الموقع)** + زر **إضافة نوع رمز** (يعمل Enter أيضًا).

## الحقول المخصصة

"حقول مستخدم بأنواع محددة تُعرض كأعمدة في الجدول ويمكن تحريرها لكل مهمة."

صف واحد لكل حقل: **الاسم** (قابل للتعديل مباشرة)، **النوع** (للقراءة فقط بعد الإنشاء) وزر إزالة.

في الأسفل: حقل إدخال **حقل جديد (مثل المقاول)**، منتقي نوع — **نص**، **رقم**، **عدد صحيح**، **تكلفة**، **تاريخ** أو **نعم/لا** — وزر **إضافة حقل** (يعمل Enter أيضًا). لا يمكن تغيير النوع بعد الإنشاء؛ أنشئ حقلًا جديدًا عند الحاجة.

## أين تظهر التعريفات

- كقسم إدخال **الرموز والحقول** لكل مهمة في لوحة الخصائص وحوار المهمة.
- كأعمدة في عرض الجدول (الحقول المخصصة) وكبُعد تجميع/تصفية (رموز الأنشطة).
