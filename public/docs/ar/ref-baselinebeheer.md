# إدارة خطوط الأساس

تدير نافذة **خطوط الأساس** اللقطات المحفوظة من الجدول: الحفظ، إعادة التسمية، اختيار خط الأساس النشط والحذف.

## الفتح

**التخطيط** ← مجموعة الشريط **خطوط الأساس والتقدم** ← **حفظ خط أساس…** أو **إدارة خطوط الأساس…** (كلاهما يفتح النافذة نفسها). **Esc** أو **إغلاق** أو علامة الإغلاق أو نقرة خارج النافذة يغلقها؛ تسري كل التغييرات في هذه النافذة فورًا.

## جدول خطوط الأساس

صف واحد لكل خط أساس محفوظ:

- **نشط** — زر اختيار؛ يمكن أن يكون خط أساس واحد فقط نشطًا. خط الأساس النشط هو أساس المقارنة لتراكب خط الأساس في مخطط جانت وتقرير الانحراف.
- **الاسم** — قابل للتعديل مباشرة في الصف.
- **تاريخ الإنشاء** — تاريخ حفظ خط الأساس.
- **حذف** (سلة مهملات) — يزيل خط الأساس. إن كان النشط، تطلب النافذة تأكيدًا أولًا ("هل تريد حذف خط الأساس النشط؟")؛ بعدها يصبح آخر خط أساس محفوظ متبقٍ نشطًا، أو لا شيء إن لم يتبقَّ شيء.

بدون خطوط أساس تعرض النافذة "لا توجد خطوط أساس بعد".

## حفظ خط أساس جديد

- **حقل الاسم** — معبَّأ مسبقًا بـ"خط الأساس {n} — {التاريخ}"؛ عدّل الاسم كما تشاء.
- **حفظ** — يسجّل بداية ونهاية (وللمعالم، تاريخ) كل مهمة ويجعل خط الأساس الجديد نشطًا.
- **تحذير** — إذا كان الجدول قديمًا منذ آخر حساب، تظهر "الجدول قديم — أعد الحساب أولاً (F5)": تلميح لا حاجز. خط أساس على جدول قديم يجمّد التواريخ الخاطئة.
