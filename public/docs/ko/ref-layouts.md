# 레이아웃 저장 및 불러오기

레이아웃은 저장된 뷰 구성입니다: 열, 그룹화, 정렬, 필터, 시간 척도를 하나의 패키지로 묶은 것입니다. 레이아웃은 앱 전역(이 기기에서)입니다 — 하나의 프로젝트 파일에 속하지 않으므로 어떤 문서에서든 사용할 수 있습니다.

## 열기

**보기** → 리본 그룹 **레이아웃**. 여기에는 저장한 레이아웃이 있는 선택기와 세 개의 버튼이 있습니다:

- **다른 이름으로 저장…**과 **관리…** — 둘 다 **레이아웃 관리** 창(아래)을 엽니다.
- **업데이트** — 선택기에서 선택한 레이아웃을 현재 뷰로 덮어씁니다; **(없음)**이 선택된 동안은 비활성화됩니다.

선택기에서 레이아웃을 선택하면 즉시 적용됩니다.

## 레이아웃 관리 창

저장된 레이아웃이 없으면 창에 "아직 저장된 레이아웃이 없습니다."라고 표시됩니다. 그렇지 않으면 레이아웃당 한 행에 다음이 있습니다:

- **이름** — 행에서 직접 편집 가능(이름 바꾸기).
- **적용**(체크 표시) — 먼저 확인을 요청합니다: "'…' 레이아웃을 적용하시겠습니까? 현재 열/그룹화/정렬/필터/스케일이 대체됩니다."
- **업데이트** — 확인 없이 레이아웃을 현재 뷰로 덮어씁니다.
- **삭제**(휴지통 아이콘) — 먼저 확인을 요청합니다.

확인은 앱 내 작은 대화상자로 나타납니다; **Esc** 또는 **취소**로 중단됩니다.

## 다른 이름으로 레이아웃 저장…

창 하단에서: **이름**을 입력하고 **저장**을 클릭하세요 — 현재 뷰가 새 레이아웃으로 저장되고 활성 레이아웃이 됩니다. 이름이 없으면 레이아웃은 기본 이름 "이름"을 갖습니다.

## 레이아웃이 담는 것

- 열(표시 여부, 순서, 너비) — [열 선택](docs://ref-kolommen)을 참조하세요.
- 그룹화 및 정렬(**보기** → **그룹화…** / **정렬…**).
- 필터 — [필터](docs://ref-filters)를 참조하세요.
- 간트 시간 척도.

포함되지 않는 것: 확대/축소 수준 세부 정보, 패널 너비, 선택 항목.
