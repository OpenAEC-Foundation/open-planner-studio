# 자원 달력

**자원 달력** 창은 하나의 자원 고유의 달력을 편집합니다 — 예를 들어 주 4일 근무하는 팀. 이 양식은 [달력 대화상자](docs://ref-kalenderdialoog)와 동일합니다; 이 문서는 차이점만 설명합니다.

## 열기

- 자원 패널 열기: **자원** → 리본 그룹 **관리** → **자원**(전체 패널) 또는 **도킹**(간트 옆에 도킹).
- 자원의 **달력** 열에서 달력을 선택하고 옆의 연필 아이콘(**편집…**)을 클릭하여 편집합니다; 같은 드롭다운에서 새 달력을 만들 수도 있습니다.

## 달력 대화상자와의 차이점

- **한 번에 하나의 달력** — 왼쪽에 라이브러리 목록이 없고, 프로젝트 기본값 별표도 없습니다; 양식만 있습니다.
- **적용**은 달력을 저장합니다; **취소**, **Esc**, 닫기 버튼 또는 창 밖 클릭은 변경 사항을 폐기합니다.
- **자동 재계산 없음** — 자원 달력은 CPM 날짜를 변경하지 않습니다; 부하(히스토그램)와 평준화에 반영되며, 이는 각각 F5 또는 **평준화…**로 직접 다시 실행합니다.

## 필드

전체 필드 참조는 [달력 대화상자](docs://ref-kalenderdialoog)를 참조하세요: **이름**, **근무일**(월–금 및 연속(24/7) 프리셋 포함), **시작 (시)** / **종료 (시)** / **일일 근무 시간**, **근무 시간** 섹션(시간 단위 계획이 켜진 경우), **공휴일 생성…**, **공휴일** 목록.
