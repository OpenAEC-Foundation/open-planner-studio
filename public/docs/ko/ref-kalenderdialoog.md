# 달력 대화상자

**달력** 창은 프로젝트의 달력 라이브러리를 관리합니다: 왼쪽에는 모든 달력 목록, 오른쪽에는 선택한 달력의 편집 양식이 있습니다.

## 열기

- **계획** → 리본 그룹 **달력** → **달력** 또는 **공휴일** 버튼.
- **설정**(리본 탭) → 리본 그룹 **달력** → **달력**.
- 프로젝트 마법사에서: 달력으로 **사용자 지정…**을 선택하면 생성 후 이 창이 열립니다.

## 적용 및 취소

새/복제/삭제를 포함한 모든 편집은 작업 사본에서 이루어집니다. **적용**(또는 **Enter**)은 모든 것을 한 번에 기록하고 일정을 다시 계산합니다; **취소**, **Esc**, 닫기 버튼 또는 창 밖 클릭은 모든 변경 사항을 폐기합니다.

## 라이브러리 (왼쪽 열)

- **목록** — 모든 달력; 별표는 **프로젝트 달력**(자체 달력이 없는 작업의 기본값)을 표시합니다.
- **+** — **새 달력**.
- **복제** — 선택한 달력의 사본.
- **삭제** — 마지막 남은 달력은 삭제할 수 없습니다; 프로젝트 기본값을 삭제하면 다른 달력이 기본값이 됩니다.
- **프로젝트 기본값으로 설정** — 선택한 달력을 프로젝트 달력으로 만듭니다(양식 위의 버튼).

## 양식 (오른쪽 열)

- **이름** — 자유 입력 이름.
- **근무일** — **월**부터 **일**까지의 버튼; 켜짐 = 근무일. 프리셋: **월–금**(표준 주, 07–16시, 8시간/일)과 **연속 (24/7)**.
- **시작 (시)** / **종료 (시)** / **일일 근무 시간** — 하루 단위의 근무 시간. 달력에 근무 시간대가 있고 시간 단위 계획이 켜지면 숨겨집니다; 그 경우 시간대가 시간을 결정합니다.

## 근무 시간 (시간 단위 계획이 켜진 경우에만)

- **산출된 시간/일** — 시간대에서 산출된 확인 수치.
- 프리셋: **주간 근무**, **2교대**, **3교대**, **야간 근무**, **24/7** — 각각 근무 시간대를 한 번에 설정합니다.
- **프리셋으로 저장…** — 현재 근무 시간을 나만의 프리셋으로(이 기기에) 저장합니다; 나만의 프리셋은 삭제 버튼이 있는 버튼으로 나타납니다.
- **요일별로 설정…** / **근무 시간 표시/숨기기** — 시간대 편집기를 열거나 접습니다.
- **시간대 편집기** — 요일별로 시간대(시작–종료) 목록, 각각 **다음 날** 체크박스(자정을 넘는 야간 근무), **구간 추가**(두 구간 사이의 간격은 휴식 시간), **모든 근무일에 복사**, 일일 시간 합계, 하단의 산출된 시간/일. [달력 및 시간 단위 계획](docs://gids-kalenders-uren)을 참조하세요.

## 공휴일 생성…

프로젝트 기간 전체에 걸쳐 규칙 기반으로 공휴일 목록을 생성합니다:

- **국가** — 네덜란드, 독일, 벨기에, 프랑스, 영국, 오스트리아, 스위스 또는 **공휴일 없음**.
- **지역** — 지역별 세트가 있는 국가에만 해당; 기본값은 **전국**.
- **건설 휴가** — 네덜란드 전용: **없음**, **북부**, **중부** 또는 **남부**; 참고용 날짜라는 힌트와 함께.
- **미리보기** — 요약 줄("공휴일 n개, 연도–연도"), 전체 목록으로 확장 가능.
- **생성**은 공휴일 목록을 대체합니다; **취소**는 블록을 닫습니다.
- 프로젝트가 이제 생성된 연도를 넘어서면 상단에 **다시 생성** 버튼과 함께 힌트가 나타납니다.

## 공휴일

목록 자체: 행마다 **설명**, **시작**, **종료**, 제거 버튼; **공휴일 추가**는 새 행을 만듭니다. 여러 날에 걸친 기간(건설 휴가, 서리 지연)은 단순히 더 긴 시작–종료 범위를 가진 하나의 행입니다.
