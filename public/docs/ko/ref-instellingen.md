# 설정

**설정** 창에는 앱 설정이 있습니다: 프로젝트 파일과 무관하게 이 기기에 적용되는 환경설정입니다. 모든 변경 사항은 즉시 적용되고 저장됩니다 — 확인 버튼이 없습니다. 계산된 일정을 바꾸는 스케줄링 옵션은 대신 프로젝트에 속합니다 — [프로젝트 정보](docs://ref-projectgegevens)를 참조하세요.

## 열기 — 세 가지 입구, 같은 내용

- 제목 표시줄의 **톱니바퀴**(⚙).
- **설정**(리본 탭) → 리본 그룹 **프로젝트** → **설정**.
- **파일** → **설정**(백스테이지).

세 곳 모두 정확히 같은 설정을 보여 주며, 세 개의 탭에 나뉘어 있습니다: **일반**, **언어**, **타임라인 / 확대/축소**.

## 일반 탭

- **테마** — **어두운 테마**, **밝은 테마** 또는 **고대비**; 카드를 클릭하여 전환합니다.
- **문서 전환 스타일** — 열린 문서 간 전환 방법: **가로 탭**, **세로 탭** 또는 **알약**.
- **날짜 형식** — **dd-mm-yyyy**, **mm-dd-yyyy** 또는 **yyyy-mm-dd**. 표시 전용이며, 파일과 계산에는 영향을 주지 않습니다.
- **버전** — 앱의 버전 번호(읽기 전용).
- **업데이트** — **업데이트 확인**은 업데이트 창을 엽니다. 업데이트 설치는 데스크톱 앱에서만 작동합니다; Snap과 AppImage 설치는 자체 채널로 업데이트됩니다.
- **기본 확대/축소** — 기본 확대/축소 수준(읽기 전용, 30 px/일).
- **디버그 터미널** — **디버그 터미널 활성화**는 문제 해결을 위한 로그 패널을 표시합니다.
- **프로젝트 정보...** — [프로젝트 정보](docs://ref-projectgegevens) 창으로 가는 바로 가기.
- **투어** — **투어 시작**은 소개 투어를 다시 재생합니다. 같은 재시작 버튼은 **보기** 리본 탭 → **투어**와 백스테이지(**파일** → **투어 시작**)에도 있습니다.

## 언어 탭

- **언어** — 앱의 표시 언어; 14개 언어, 즉시 적용됩니다.

## 타임라인 / 확대/축소 탭

- **시간 단위 계획** — **시간 단위 계획 사용**은 시간/분 단위 스케줄링을 켭니다: 시간 단위 시간축, 근무 시간대가 있는 교대, 시간 단위로 정확한 작업 막대. 끄면 ⇒ 앱은 완전히 일 단위로 유지됩니다. 스위치가 켜지면 **일/시간 혼합 계획 허용**이 나타납니다(하나의 프로젝트에 일 단위와 시간 단위 작업). 스위치가 꺼진 상태에서 시간 단위 계획이 포함된 파일을 열면 상단에 **시간 단위 계획 사용**을 제공하는 바가 나타납니다. [달력 및 시간 단위 계획](docs://gids-kalenders-uren)을 참조하세요.
- **기간 표시** — **자동 (작업별 고유 단위)**, **항상 일** 또는 **항상 시간**.
- **중단 시 작업 막대** — **분할 안 함**, **선택 시 분할** 또는 **항상 분할**: 비근무일 주변에서 막대가 시각적으로 분할되는지 여부.
- **주 시작 요일** — **월요일** 또는 **일요일**(시간축의 주 레이아웃).
- **최대 확대 시 15분 단위 표시** — 시간 시간축에 추가적인 15분 단위 눈금.
- **계산** — **자동 계산**은 F5를 기다리지 않고 일정이 오래되는 즉시 다시 계산합니다.
- **스크롤 및 확대/축소** — **모드**:
- **위치** — 커서 위치가 스크롤 방향을 결정합니다; **화면 분할**(**좌/우**, **상/하** 또는 **우상단 모서리**) 포함. Ctrl+스크롤 = 확대/축소, Shift+스크롤 = 가로.
- **키** — 어떤 조작(**스크롤**, **Ctrl + 스크롤**, **Shift + 스크롤**)이 어떤 기능(**세로**, **가로**, **확대/축소**)을 수행하는지 칩을 드래그하여 지정합니다; 이미 사용 중인 슬롯에 놓으면 조작이 서로 바뀝니다.
- **확대/축소 + 드래그** — 스크롤 휠로 확대/축소(커서 기준); 차트 배경을 드래그하여 보기를 이동합니다.
