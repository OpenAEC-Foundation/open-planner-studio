# 열 선택

**열** 창은 테이블 탭이 어떤 열을, 어떤 순서로, 얼마나 넓게 표시할지 제어합니다. (간트 왼쪽의 작업 테이블은 고정 열을 가집니다: WBS, 작업 이름, 기간.)

## 열기

**보기** → 리본 그룹 **표시** → **열…**. 모든 변경 사항은 즉시 적용됩니다 — 별도의 확인 단계는 없습니다; **닫기**, **Esc**, 닫기 버튼 또는 창 밖 클릭으로 닫힙니다.

## 선택된 열

열당 한 행, 다음을 포함:

- **드래그 핸들** — 행을 드래그하여 열 순서를 변경합니다.
- **표시** — 체크 해제하면 목록에서 제거하지 않고 열을 숨깁니다.
- **이름** — 테이블이 표시하는 필드 라벨.
- **너비** — 픽셀 단위(최소 40).

## 사용 가능한 필드

선택된 열 아래에는 **사용 가능한 필드** 목록이 있습니다: 아직 열이 아닌 모든 필드. 하나를 클릭하면 열로 추가됩니다. 표준 필드 외에도 분석 필드인 **마일스톤**, **자유 여유**, **간섭 여유**, **준임계**, **여유 경로**, 그리고 **자원**과 프로젝트의 작업 코드 및 사용자 정의 필드가 있습니다. 세 가지 여유 필드와 여유 경로는 일치하는 스케줄링 옵션으로 계산한 후에만 값을 얻습니다 — [크리티컬 패스 및 고급 분석](docs://gids-kritiek-pad-analyse)을 참조하세요.

## 기본값으로 재설정

**기본값으로 재설정**은 표준 열 구성을 복원합니다; 추가로 더해진 필드는 사용 가능한 필드 목록으로 다시 이동합니다.

열 구성은 저장된 레이아웃의 일부입니다 — [레이아웃 저장 및 불러오기](docs://ref-layouts)를 참조하세요.
