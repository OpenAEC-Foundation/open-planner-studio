# 키보드 단축키 참조

항상 최신 상태인 전체 단축키 목록은 앱 자체에 있습니다: **키보드 단축키** 창. 이 문서는 의도적으로 그 목록을 중복하지 않습니다 — 창은 앱에서 직접 만들어지므로 결코 뒤처지지 않습니다.

## 열기

- **Ctrl+/**(macOS: **Cmd+/**).
- **보기** → 리본 그룹 **키보드 단축키** → **키보드 단축키**.

## 표시되는 내용

목록은 다섯 개의 카테고리로 그룹화됩니다:

- **파일** — 계산, 저장, 열기, 새 프로젝트.
- **편집** — 복사/붙여넣기, 실행 취소, 삭제, 선택.
- **구조** — 들여쓰기/내어쓰기, 삽입, 마일스톤, 작업 이동.
- **보기** — 프레젠테이션, 확대/축소, 이 창 자체.
- **탐색** — 문서 전환, 오늘로 이동, 도움말 열기(F1).

작업에 여러 키 조합이 있으면 한 행에 함께 표시됩니다("또는"). macOS에서는 창이 자동으로 **Ctrl** 대신 **Cmd**를 표시합니다.
