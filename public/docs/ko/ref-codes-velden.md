# 코드 및 필드 (구조 대화상자)

**코드 및 필드** 창은 프로젝트 구조 정의를 관리합니다: **작업 코드**(위치나 공종 같은 자유롭게 정의 가능한 차원)와 **사용자 정의 필드**(형식이 지정된 사용자 필드). 작업별 값은 이후 속성 패널이나 [작업 대화상자](docs://ref-taakdialoog)를 통해 채워 넣습니다.

## 열기

**계획** → 리본 그룹 **구조** → **코드 및 필드**. **Esc**, 닫기 버튼 또는 창 밖 클릭으로 닫힙니다. 모든 변경 사항은 즉시 적용되며(Ctrl+Z로 실행 취소 가능) — 별도의 저장 버튼은 없습니다.

## 작업 코드

"그룹화와 필터링을 위한 자유롭게 정의 가능한 차원(예: 위치, 공종) — 작업당 유형별로 값은 최대 하나."

코드 유형당 한 블록:

- **코드 유형 이름** — 직접 편집 가능.
- **코드 유형 제거**(휴지통) — 모든 값과 작업에 대한 할당을 포함하여 유형을 제거합니다.
- 값당 한 행: **코드**(짧은 라벨), **설명**, **색상** 선택기(그룹 등에 색상을 지정), 그리고 제거 버튼.
- **값 추가** — 이 유형 아래에 새 값을 추가합니다.

하단에는 입력 필드 **새 코드 유형 (예: 위치)** + 버튼 **코드 유형 추가**(Enter도 작동)가 있습니다.

## 사용자 정의 필드

"테이블에 열로 표시되고 작업별로 편집 가능한, 형식이 지정된 사용자 필드."

필드당 한 행: **이름**(직접 편집 가능), **유형**(생성 후에는 읽기 전용), 제거 버튼.

하단에는 입력 필드 **새 필드 (예: 시공사)**, 유형 선택기 — **텍스트**, **숫자**, **정수**, **비용**, **날짜** 또는 **예/아니오** — 및 **필드 추가** 버튼(Enter도 작동)이 있습니다. 유형은 생성 후 변경할 수 없습니다; 필요하면 새 필드를 만드세요.

## 정의가 표시되는 위치

- 속성 패널과 작업 대화상자의 작업별 **코드 및 필드** 항목 섹션으로.
- 테이블 뷰의 열(사용자 정의 필드)과 그룹화/필터 차원(작업 코드)으로.
