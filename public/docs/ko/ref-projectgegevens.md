# 프로젝트 정보

**프로젝트 정보** 창에는 프로젝트 메타데이터와 스케줄링 옵션이 있는 **계산** 섹션이 있습니다. 같은 양식은 **새로 만들기**의 프로젝트 마법사 역할도 합니다.

## 열기

- **설정**(리본 탭) → 리본 그룹 **프로젝트** → **프로젝트 정보**.
- 설정 창(톱니바퀴 ⚙) → **일반** 탭 → **프로젝트 정보...**
- **파일** → **프로젝트 정보** — 메타데이터 필드만 있는 백스테이지의 단순화된 버전(계산 섹션 없음).

**적용**은 모든 변경 사항을 한 번에 커밋합니다; **취소**, **Esc** 또는 창 밖 클릭은 이를 폐기합니다. **Enter**는 적용과 동일하게 작동합니다.

## 메타데이터

- **프로젝트 이름** — 제목 표시줄과 문서 탭에 표시되는 이름.
- **설명** — 자유 텍스트.
- **엔지니어** 및 **회사** — 자유 텍스트; IFC 파일에 저장됩니다.
- **시작일** — 계산의 기준이 되는 프로젝트 시작일.
- **종료일** — 정보 제공용 프로젝트 종료일.

## 계산

이 프로젝트의 스케줄링 옵션 — 앱이 아닌 파일에 저장되므로 다른 컴퓨터로 이동해도 함께 유지됩니다. 여기서 무언가를 변경하면 **적용** 후 일정이 자동으로 다시 계산됩니다.

- **크리티컬 정의** — **총 여유 ≤ 임계값**(**임계값(작업일)**과 함께, 기본값 0) 또는 **최장 경로**.
- **여유 계산** — **최소(시작/종료)**(기본값), **시작 여유** 또는 **종료 여유**.
- **끝이 열린 작업을 크리티컬로 처리** — 후속 작업이 없는 작업을 크리티컬로 표시합니다.
- **준임계로 표시** — 체크하면 추가로 **임계값**(기본값 2 작업일; 단위는 기간 표시를 따르므로 시간일 수도 있음)이 나타납니다: 여유가 적은 작업에 "준임계" 표시가 붙습니다.
- **다중 여유 경로** — 체크하면 **방법**(**자유 여유(피일링)** 또는 **총 여유(순위)**)과 **최대 경로 수**(기본값 10)가 나타납니다: 계산이 가장 중요한 여유 경로에 번호를 매깁니다.
- **지연 캘린더** — 관계의 지연을 계산할 때 사용하는 달력: **선행**(기본값), **후행**, **24시간** 또는 **프로젝트 캘린더**.

이러한 결과를 읽는 방법은 [크리티컬 패스 및 고급 분석](docs://gids-kritiek-pad-analyse)에서 다룹니다.

## 프로젝트 마법사 (새로 만들기)

**새로 만들기**는 같은 창을 마법사로 엽니다(제목 **새 프로젝트**, 버튼 **만들기**). 메타데이터 필드 외에 마법사에는 다음이 있습니다:

- **공정 단계 템플릿** — **비어 있음**, **주택 건설** 또는 **비주거용 건축 / 리노베이션**: 단계 구조로 새 프로젝트를 채웁니다.
- **교대** — 시간 단위 계획이 켜진 경우에만 표시됨: **주간 근무**(기본값), **2교대**, **3교대** 또는 **24/7**.
- **공휴일 세트** — 프로젝트 달력을 생성합니다: 국가를 선택하고(해당하는 경우 지역과 건설 휴가와 함께), **공휴일 없음**, 또는 **사용자 지정…** — 후자는 생성 직후 달력 대화상자를 열어 달력을 직접 구성할 수 있게 합니다. [달력 대화상자](docs://ref-kalenderdialoog)를 참조하세요.

계산 섹션은 마법사에는 없습니다; 이후 위의 입구 중 하나를 통해 설정하세요.
