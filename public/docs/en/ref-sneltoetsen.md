# Keyboard shortcut reference

The complete, always up-to-date shortcut list lives in the app itself: the **Keyboard Shortcuts** window. This documentation deliberately does not duplicate that list — the window is built straight from the app, so it can never fall behind.

## Opening

- **Ctrl+/** (macOS: **Cmd+/**).
- **View** → ribbon group **Keyboard Shortcuts** → **Keyboard Shortcuts**.

## What you see

The list is grouped into five categories:

- **File** — calculate, save, open, new project.
- **Edit** — copy/paste, undo, delete, select.
- **Structure** — indent/outdent, insert, milestone, moving tasks.
- **View** — presentation, zoom, this window itself.
- **Navigation** — switching documents, jumping to today, opening Help (F1).

If an action has several key combinations, they share one row ("or"). On macOS the window automatically shows **Cmd** instead of **Ctrl**.
