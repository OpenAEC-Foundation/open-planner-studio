# Référence des raccourcis clavier

La liste complète et toujours à jour des raccourcis se trouve dans l'application elle-même : la fenêtre **Raccourcis clavier**. Cette documentation ne duplique délibérément pas cette liste — la fenêtre est construite directement à partir de l'application, elle ne peut donc jamais prendre du retard.

## Ouverture

- **Ctrl+/** (macOS : **Cmd+/**).
- **Affichage** → groupe de ruban **Raccourcis clavier** → **Raccourcis clavier**.

## Ce que vous voyez

La liste est regroupée en cinq catégories :

- **Fichier** — calculer, enregistrer, ouvrir, nouveau projet.
- **Édition** — copier/coller, annuler, supprimer, sélectionner.
- **Structure** — mettre/réduire en retrait, insérer, jalon, déplacer des tâches.
- **Affichage** — présentation, zoom, cette fenêtre elle-même.
- **Navigation** — changer de document, aller à aujourd'hui, ouvrir l'Aide (F1).

Si une action a plusieurs combinaisons de touches, elles partagent une ligne (« ou »). Sur macOS, la fenêtre affiche automatiquement **Cmd** au lieu de **Ctrl**.
