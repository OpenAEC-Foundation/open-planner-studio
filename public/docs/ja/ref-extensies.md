# 拡張機能の管理とインストール

拡張機能は、追加のインポート形式やカスタムのリボンボタンなど、アプリに機能を追加します。これらはアプリレベルのものです。プロジェクトファイルではなく、この端末のこのインストールに属します。

## 開く

**ファイル** → **拡張機能**（Backstage）。上部には **インストール済み** と **参照** の2つのタブが、**ZIP** と **JS** ボタンの横にあり、その下に検索フィールド（**拡張機能を検索...**）があります。

## インストール済み

拡張機能ごとに1枚のカードがあり、名前、バージョン、カテゴリ、説明、作者に加えて、次の項目があります。

- **オン/オフの切り替え** — 削除せずに拡張機能を有効化・無効化します。
- **削除** — もう一度 **確認** をクリックすると完全に削除されます。

読み込みに失敗した拡張機能は、そのカードにエラーメッセージを表示します。拡張機能がない場合、タブには「拡張機能はまだインストールされていません。」と表示されます。

## 参照（カタログ）

**参照** タブは、オンラインの拡張機能カタログを取得します（インターネット接続が必要です）。カタログの各項目は **インストール** ボタン付きのカードです。すでにインストール済みの拡張機能には **インストール済み** バッジが表示されます。読み込みに失敗した場合は、**再試行** 付きのエラーメッセージが表示されます。

## ファイルからインストールする

- **ZIP** — （`manifest.json` ＋ `main.js` を含む）拡張機能の ZIP をインストールします。
- **JS** — マニフェストが埋め込まれた単一の `.js` ファイルをインストールします。

インストール後、拡張機能は即座に有効化され、リボンボタンがあればすぐに表示されます。

## 拡張機能を通じたインポート

**ファイル** → **インポート** には、インストール済みの拡張機能が提供するインポート形式が一覧表示されます。形式をクリックしてファイルを選んでください。インポート用の拡張機能がない場合、ページには「インポート用拡張機能がありません。拡張機能から追加してください。」と表示されます。組み込みのインポート形式（CSV、MS Project、P6）はこれとは別です — [インポート/エクスポート](docs://gids-import-export) を参照してください。

## 自分で拡張機能を書く

拡張機能の作者向けガイド（マニフェスト、API、権限）は、リポジトリにあります：`github.com/OpenAEC-Foundation/open-planner-studio`、ファイル `docs/extensions.md`。
