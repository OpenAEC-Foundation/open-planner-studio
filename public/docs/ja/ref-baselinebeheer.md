# ベースライン管理

**ベースライン** ウィンドウは、保存されたスケジュールのスナップショットを管理します。保存、名前の変更、アクティブなベースラインの選択、削除です。

## 開く

**計画** → **ベースラインと進捗** リボングループ → **ベースラインを保存…** または **ベースラインを管理…**（どちらも同じウィンドウを開きます）。**Esc**、**閉じる**、閉じるボタン、またはウィンドウの外側のクリックで閉じます。このウィンドウでのすべての変更は即座に反映されます。

## ベースラインの表

保存されたベースラインごとに1行：

- **アクティブ** — ラジオボタン。アクティブにできるベースラインはちょうど1つです。アクティブなベースラインが、ガントのベースラインオーバーレイと差異レポートの比較基準になります。
- **名前** — 行内で直接編集できます。
- **作成日** — ベースラインが保存された日付。
- **削除**（ゴミ箱）— ベースラインを削除します。それがアクティブなベースラインの場合、ウィンドウは先に確認を求めます（「アクティブなベースラインを削除しますか？」）。その後、残っているベースラインのうち最後に保存されたものがアクティブになります。何も残っていない場合はどれもアクティブになりません。

ベースラインがない場合、ウィンドウには「ベースラインはまだありません」と表示されます。

## 新しいベースラインを保存

- **名前フィールド** — 「ベースライン {n} — {date}」で事前に入力されています。必要に応じて名前を調整してください。
- **保存** — すべてのタスクの開始日、終了日、（マイルストーンの場合は）日付を記録し、新しいベースラインをアクティブにします。
- **警告** — 最後の計算以降スケジュールが古くなっている場合、「スケジュールが古くなっています — 先に再計算してください（F5）」と表示されます。これはヒントであり、ブロックはしません。古いスケジュールに対するベースラインは、間違った日付を凍結してしまいます。
