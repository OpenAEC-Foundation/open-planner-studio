# キーボードショートカット一覧

完全で常に最新のショートカット一覧は、アプリ自体の中にあります：**キーボードショートカット** ウィンドウです。このドキュメントでは、あえてその一覧を複製していません — このウィンドウはアプリから直接構築されているため、決して古くなることがないからです。

## 開く

- **Ctrl+/**（macOS：**Cmd+/**）。
- **表示** → **キーボードショートカット** リボングループ → **キーボードショートカット**。

## 表示内容

一覧は5つのカテゴリにグループ化されています。

- **ファイル** — 計算、保存、開く、新しいプロジェクト。
- **編集** — コピー／貼り付け、元に戻す、削除、選択。
- **構造** — インデント／インデント解除、挿入、マイルストーン、タスクの移動。
- **表示** — プレゼンテーション、ズーム、このウィンドウ自体。
- **ナビゲーション** — ドキュメントの切り替え、今日へのジャンプ、ヘルプを開く（F1）。

1つの操作に複数のキーの組み合わせがある場合は、1つの行にまとめて表示されます（「または」）。macOS では、ウィンドウは自動的に **Ctrl** の代わりに **Cmd** を表示します。
