# カレンダーダイアログ

**カレンダー** ウィンドウは、プロジェクトのカレンダーライブラリを管理します。左側にすべてのカレンダーのリスト、右側に選択したカレンダーの編集フォームがあります。

## 開く

- **計画** → **カレンダー** リボングループ → **カレンダー** または **祝日** ボタン。
- **設定**（リボンタブ）→ **カレンダー** リボングループ → **カレンダー**。
- プロジェクトウィザードから：カレンダーとして **カスタム…** を選ぶと、作成後にこのウィンドウが開きます。

## 適用とキャンセル

すべての編集 — 新規／複製／削除を含む — は作業用コピー上で行われます。**適用**（または **Enter**）はすべてを一度に書き込み、スケジュールを再計算します。**キャンセル**、**Esc**、閉じるボタン、またはウィンドウの外側のクリックは、すべての変更を破棄します。

## ライブラリ（左の列）

- **リスト** — すべてのカレンダー。星印は **プロジェクトカレンダー**（独自のカレンダーを持たないタスクの既定）を示します。
- **＋** — **新規カレンダー**。
- **複製** — 選択したカレンダーのコピー。
- **削除** — 最後の1つのカレンダーには使えません。プロジェクトの既定カレンダーを削除すると、別のカレンダーが既定になります。
- **プロジェクトの既定に設定** — 選択したカレンダーをプロジェクトカレンダーにします（フォームの上のボタン）。

## フォーム（右の列）

- **名前** — 自由な名前。
- **稼働日** — **月** から **日** までのボタン。オン＝稼働日。プリセット：**月〜金**（標準の週、7〜16時、8時間/日）と **常時（24/7）**。
- **開始（時）** / **終了（時）** / **1日の時間数** — 1日全体の勤務時間。カレンダーに稼働時間帯があり、時間計画がオンの場合は非表示になります。その場合は時間帯が時刻を決定します。

## 稼働時間（時間計画が有効な場合のみ）

- **算出された時間/日** — 時間帯から導かれる確認用の数値。
- プリセット：**日勤**、**2交代**、**3交代**、**夜勤**、**24/7** — それぞれが稼働時間帯を一度に設定します。
- **プリセットとして保存…** — 現在の稼働時間を（この端末上の）自分専用のプリセットとして保存します。自分のプリセットは削除用の×印付きのボタンとして表示されます。
- **曜日ごとに設定…** / **稼働時間を表示/非表示** — 時間帯エディターを開く、または折りたたみます。
- **時間帯エディター** — 曜日ごとに時間帯（開始〜終了）のリストがあり、それぞれに **翌日** チェックボックス（深夜0時をまたぐ夜勤用）、**帯を追加**（2つの帯の間の空きは休憩）、**すべての稼働日にコピー**、日ごとの合計時間、下部に算出された時間/日が表示されます。[カレンダーと時間計画](docs://gids-kalenders-uren) を参照してください。

## 祝日を生成…

プロジェクト期間全体にわたって、ルールベースで祝日リストを生成します。

- **国** — オランダ、ドイツ、ベルギー、フランス、イギリス、オーストリア、スイス、または **祝日なし**。
- **地域** — 地域別のセットがある国のみ。既定は **全国**。
- **建設業休暇** — オランダのみ：**なし**、**北部**、**中部**、**南部**。これらが目安の日付であるというヒント付きです。
- **プレビュー** — 要約行（「n 件の祝日、年〜年」）。展開すると全リストになります。
- **生成** は祝日リストを置き換えます。**キャンセル** はこのブロックを閉じます。
- プロジェクトが生成済みの年を超えて続く場合、上部に **再生成** ボタン付きのヒントが表示されます。

## 祝日

リストそのもの：各行に **説明**、**開始**、**終了**、削除ボタン。**祝日を追加** で新しい行が作成されます。複数日にわたる期間（建設業休暇、霜による作業中断）は、単に開始〜終了の幅が長い行にすぎません。
