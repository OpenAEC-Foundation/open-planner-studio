# 設定

**設定** ウィンドウには、アプリの設定が入っています。プロジェクトファイルとは無関係な、この端末に適用される好みの設定です。すべての変更は即座に適用・保存されます — OK ボタンはありません。計算されるスケジュールを変更するスケジューリングオプションは、代わりにプロジェクトに属します — [プロジェクト情報](docs://ref-projectgegevens) を参照してください。

## 開く — 3つの入口、同じ内容

- タイトルバーの **歯車**（⚙）。
- **設定**（リボンタブ）→ **プロジェクト** リボングループ → **設定**。
- **ファイル** → **設定**（Backstage）。

3つとも、まったく同じ設定を、**一般**、**言語**、**タイムライン／ズーム** の3つのタブに分けて表示します。

## 一般タブ

- **テーマ** — **ダーク**、**ライト**、**ハイコントラスト**。カードをクリックして切り替えます。
- **ドキュメント切り替えスタイル** — 開いているドキュメント間をどう切り替えるか：**横タブ**、**縦タブ**、**ピル**。
- **日付形式** — **dd-mm-yyyy**、**mm-dd-yyyy**、**yyyy-mm-dd**。表示のみに影響し、ファイルや計算には影響しません。
- **バージョン** — アプリのバージョン番号（読み取り専用）。
- **アップデート** — **アップデートを確認** でアップデートウィンドウを開きます。アップデートのインストールはデスクトップアプリでのみ機能します。Snap と AppImage のインストールは、それぞれ独自のチャンネルで更新されます。
- **既定のズーム** — 既定のズームレベル（読み取り専用、30 px/日）。
- **デバッグターミナル** — **デバッグターミナルを有効にする** で、トラブルシューティング用のログパネルを表示します。
- **プロジェクト情報...** — [プロジェクト情報](docs://ref-projectgegevens) ウィンドウへのショートカット。
- **ツアー** — **ツアーを開始** で紹介ツアーをもう一度再生します。同じ再開ボタンは、**表示** リボンタブの **ツアー** と、Backstage（**ファイル** → **ツアーを開始**）にもあります。

## 言語タブ

- **言語** — アプリの表示言語。14言語あり、即座に適用されます。

## タイムライン／ズームタブ

- **時間計画** — **時間計画を有効にする** で、時間・分単位のスケジューリングをオンにします：時間のタイムスケール、稼働時間帯を持つシフト、時間単位で正確なタスクバー。オフの場合 ⇒ アプリは完全に日単位のままです。スイッチがオンだと、**日単位と時間単位の混在計画を許可する**（1つのプロジェクトに日単位のタスクと時間単位のタスクを混在させる）が表示されます。スイッチがオフの状態で時間計画を含むファイルを開くと、上部のバーに **時間計画を有効にする** が案内表示されます。[カレンダーと時間計画](docs://gids-kalenders-uren) を参照してください。
- **期間の表示** — **自動（タスクごとの単位）**、**常に日**、**常に時間**。
- **中断時のタスクバー** — **分割しない**、**選択時に分割**、**常に分割**：非稼働日の周りでバーを視覚的に分割するかどうか。
- **週の開始日** — **月曜日** または **日曜日**（タイムスケールの週のレイアウト）。
- **拡大時に15分単位を表示する** — 時間のタイムスケールに、追加の15分刻みの目盛りを表示します。
- **計算** — **自動計算** は、F5 を待たずに、スケジュールが古くなった時点で自動的に再計算します。
- **スクロールとズーム** — **モード**：
- **位置** — カーソルの位置がスクロール方向を決めます。**画面分割**（**左右**、**上下**、**右上隅**）付き。Ctrl＋スクロール＝ズーム、Shift＋スクロール＝水平。
- **キー** — チップをドラッグして、どの操作（**スクロール**、**Ctrl＋スクロール**、**Shift＋スクロール**）にどの機能（**垂直**、**水平**、**ズーム**）を割り当てるかを設定します。すでに使われているスロットにドロップすると、操作が入れ替わります。
- **ズーム＋ドラッグ** — スクロールホイールでズーム（カーソル基点）、チャート背景をドラッグして表示をスクロールします。
