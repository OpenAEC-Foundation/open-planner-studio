# タスクダイアログ

**タスクの編集** ウィンドウには、1つのタスクのすべてのプロパティが表示されます — 右側のプロパティパネルと同じフィールドとセクションですが、明示的な保存操作があるウィンドウの形式です。

## 開く

- ガント上のタスクを **ダブルクリック**。
- タスクを選択した状態で **F2**。
- タスクを **右クリック** → **編集...**。

## 保存とキャンセル

- **保存** はすべてのフィールドの変更を一度に適用します。名前が空の間はボタンが無効になります。**Enter** は（複数行のテキストボックス内を除いて）保存と同じ動作をします。
- **キャンセル**、**Esc**、閉じるボタン、またはウィンドウの外側のクリックは、フィールドの変更を適用せずに閉じます。
- 例外：**依存関係**、**割り当て**、**コードとフィールド** の各セクションはスケジュールに直接作用します（パネルと同一）— そこでの変更は、後でキャンセルしても即座に有効になります。

## フィールド

- **名前 \*** — 必須。ダイアログが開くと自動的にフォーカスが当たります。
- **WBS コード** — 自由入力。WBS 自動採番（計画 → 構造）がオンの場合、フィールドはロックされます。コードはアプリが管理します。
- **説明** — 自由記述のテキスト。
- **種別** — タスクの種別（例えば建設）。バーの色分けを決定します。
- **カレンダー** — **プロジェクトカレンダー**、またはライブラリ内の特定のカレンダー。このタスクの稼働日を決定します。
- **親タスク** — タスクを別の親の下に移動するか、**－ なし（ルート）－** を選びます。このフィールドはダイアログにのみ存在します。パネルでは、構造の変更はドラッグまたはインデント／インデント解除で行います。

## メモ

タスクごとのチェックリストです。各行には **完了チェックボックス**、テキストボックス、削除ボタンがあります。**メモを追加** で新しい行が作成されます。完了した行には取り消し線が引かれます。[計画と WBS](docs://gids-plannen-wbs) を参照してください。

## マイルストーン

- **マイルストーン** — チェックすると期間が0になり、バーの代わりにダイヤモンドが表示されます。
- **マイルストーンの種類** — **自動**、**開始マイルストーン**、**完了マイルストーン**。
- **必須（契約上）** — そのマイルストーンを契約上のものとしてマークします。

## 時間

- **開始日** — 計算された最早開始日を表示します。手動で変更すると、その新しい日付が計画上の開始として固定されます。
- **期間（稼働日）** — 整数の稼働日。マイルストーンでは無効化されます。
- **時間計画が有効** で、タスクに時間カレンダーが設定されている場合、同期する3つのボックスが表示されます：**日数**、**時間**、**合計時間**（いずれも整数のみ）。時間カレンダーがない場合は「時間入力には時間カレンダー（稼働時間）が必要です。」というヒントが表示されます。[カレンダーと時間計画](docs://gids-kalenders-uren) を参照してください。

## Hammock（派生期間）

サブタスクを持たず、マイルストーンでもないタスクにのみ存在します。チェックすると期間が派生値になります。**開始ドライバー**（受け取る FS/SS 関係）と **終了ドライバー**（受け取る FF/SF 関係）の間隔で、どちらも読み取り専用で表示されます。終了ドライバーがない場合、ダイアログは期間がゼロ長に戻ることを報告します。[クリティカルパスと高度な分析](docs://gids-kritiek-pad-analyse) を参照してください。

## 制約と期限

- **制約** — できるだけ早く (ASAP)、できるだけ遅く (ALAP)、指定日以降に開始 (SNET)、指定日以前に開始 (SNLT)、指定日以降に終了 (FNET)、指定日以前に終了 (FNLT)、指定日に開始 (MSO)、または指定日に終了 (MFO)。該当する場合は **制約日** を伴います。
- **必須（ピン留めロジック）** — MSO/MFO のみ：日付をハードピンし、関係のロジックを上書きします。違反は上流側で負のフロートになります。
- **副制約** — **副日付** を伴う2つ目の境界（SNET/FNET/SNLT/FNLT）。ハードピンとは併用できません。許可されない組み合わせは理由とともに赤で表示されます。
- **期限** — 計算の外にある目標日です。守れなくても警告が出るだけで、ずれは生じません。[関係と制約](docs://gids-relaties-constraints) を参照してください。

## 進捗

- **進捗（%）** — 0〜100%のスライダー。
- **実績開始** / **実績終了** — 記録された事実。マイルストーンでは単一の **実績日** フィールドになります。状態日より後の日付は拒否されます。
- **残り（稼働日）** — 読み取り専用。期間 ×（1 − 進捗）から算出されます。[ベースラインと進捗](docs://gids-baselines-voortgang) を参照してください。

## CPM 結果（読み取り専用）

**最早開始/終了**、**最遅開始/終了**、**トータルフロート**、**フリーフロート**、（計算された場合）**干渉フロート**、**クリティカルパス**（はい/いいえ）。計算（F5）の後に入力されます。

## 依存関係

このタスクのすべての関係：方向（→ 後続タスク、← 先行タスク）、相手のタスク、**駆動関係** を示す稲妻アイコン、関係タイプ（FS/SS/FF/SF）、**ラグ**（例：2d、3ed、50%）、削除ボタン。変更は即座に反映されます。

## 割り当て

割り当てられたリソースごとに：名前、**単位/日**、**曲線**、**移動先…**（割り当てを別のタスクへ移動）、削除。下部に **リソースを割り当てる**。マイルストーンやサマリータスクでは設定できません。即座に反映されます。[リソース、ヒストグラム、山崩し](docs://gids-resources-histogram) を参照してください。

## コードとフィールド

プロジェクトにアクティビティコードタイプまたはカスタムフィールドがある場合にのみ表示されます：コードタイプごとの値ピッカー、カスタムフィールドごとの型付き入力。即座に反映されます。定義は構造ダイアログで管理します — [コードとフィールド](docs://ref-codes-velden) を参照してください。
