# Referência de atalhos de teclado

A lista completa e sempre atualizada de atalhos vive na própria aplicação: a janela **Atalhos de teclado**. Esta documentação não duplica deliberadamente essa lista — a janela é construída diretamente a partir da aplicação, por isso nunca pode ficar desatualizada.

## Abrir

- **Ctrl+/** (macOS: **Cmd+/**).
- **Visualização** → grupo do friso **Atalhos de teclado** → **Atalhos de teclado**.

## O que vê

A lista está agrupada em cinco categorias:

- **Ficheiro** — calcular, guardar, abrir, novo projeto.
- **Editar** — copiar/colar, desfazer, eliminar, selecionar.
- **Estrutura** — indentar/diminuir indentação, inserir, marco, mover tarefas.
- **Visualização** — apresentação, zoom, esta própria janela.
- **Navegação** — mudar de documento, saltar para hoje, abrir Ajuda (F1).

Se uma ação tem várias combinações de teclas, partilham uma linha ("ou"). No macOS a janela mostra automaticamente **Cmd** em vez de **Ctrl**.
