# Tastenkombinationen-Referenz

Die vollständige, stets aktuelle Liste der Tastenkombinationen lebt in der App selbst: das Fenster **Tastenkombinationen**. Diese Dokumentation dupliziert diese Liste bewusst nicht — das Fenster wird direkt aus der App aufgebaut, kann also nie hinterherhinken.

## Öffnen

- **Ctrl+/** (macOS: **Cmd+/**).
- **Ansicht** → Menübandgruppe **Tastenkombinationen** → **Tastenkombinationen**.

## Was Sie sehen

Die Liste ist in fünf Kategorien gruppiert:

- **Datei** — berechnen, speichern, öffnen, neues Projekt.
- **Bearbeiten** — kopieren/einfügen, rückgängig, löschen, auswählen.
- **Struktur** — einrücken/ausrücken, einfügen, Meilenstein, Aufgaben verschieben.
- **Ansicht** — Präsentation, Zoom, dieses Fenster selbst.
- **Navigation** — Dokumente wechseln, zu heute springen, Hilfe öffnen (F1).

Hat eine Aktion mehrere Tastenkombinationen, teilen sie sich eine Zeile („oder"). Unter macOS zeigt das Fenster automatisch **Cmd** statt **Ctrl**.
