# Sneltoetsenoverzicht

完整且始终最新的快捷键列表存在于应用本身之中：**键盘快捷键**窗口。本文档刻意不重复该列表——该窗口直接由应用构建而成，因此永远不会过时。

## 打开方式

- **Ctrl+/**（macOS：**Cmd+/**）。
- **视图** → **键盘快捷键**功能区组 → **键盘快捷键**。

## 您将看到什么

列表分为五个类别：

- **文件** — 计算、保存、打开、新建项目。
- **编辑** — 复制/粘贴、撤销、删除、选择。
- **结构** — 缩进/取消缩进、插入、里程碑、移动任务。
- **视图** — 演示、缩放、此窗口本身。
- **导航** — 切换文档、跳转到今天、打开帮助（F1）。

如果某个操作有多个按键组合，它们会共用一行（“或”）。在 macOS 上，窗口会自动显示 **Cmd** 而非 **Ctrl**。
