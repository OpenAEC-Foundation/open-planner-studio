# Layouts opslaan/laden

布局是一份已保存的视图配置：列、分组、排序、筛选和时间刻度打包在一起。布局是应用全局的（在本设备上）——它们不属于单一项目文件，因此您可以在任何文档中使用它们。

## 打开方式

**视图** → **布局**功能区组。其中有一个包含您布局的选择器和三个按钮：

- **另存为…**和**管理…** — 都会打开下方的**管理布局**窗口。
- **更新** — 用当前视图覆盖选择器中所选的布局；选中**(无)**时不可用。

在选择器中选择一个布局会立即应用它。

## 管理布局窗口

没有已保存的布局时，窗口显示“尚无已保存的布局。”否则，每个布局一行，包含：

- **名称** — 可直接在行内编辑（重命名）。
- **应用**（对勾） — 先要求确认：“应用布局“……”？这将替换当前的列/分组/排序/筛选/比例。”
- **更新** — 用当前视图覆盖该布局，无需确认。
- **删除**（垃圾桶图标） — 先要求确认。

确认以应用内的小对话框形式出现；**Esc**或**取消**会中止操作。

## 另存布局为…

在窗口底部：输入一个**名称**并点击**保存** — 当前视图会作为新布局存储并成为活动布局。若未输入名称，布局会得到默认名称“名称”。

## 布局包含什么

- 列（可见性、顺序、宽度）——参见 [Kolommen kiezen](docs://ref-kolommen)。
- 分组和排序（**视图** → **分组…** / **排序…**）。
- 筛选器——参见 [Filters](docs://ref-filters)。
- 甘特图时间刻度。

不包含：缩放级别细节、面板宽度和选择状态。
