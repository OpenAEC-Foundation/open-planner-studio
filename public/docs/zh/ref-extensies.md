# Extensies beheren en installeren

扩展为应用添加功能，例如额外的导入格式或自定义功能区按钮。它们是应用级别的：属于本设备上的这次安装，而不属于某个项目文件。

## 打开方式

**文件** → **扩展**（Backstage）。顶部有两个标签——**已安装**和**浏览**——旁边是 **ZIP** 和 **JS** 按钮，下方是搜索框（**搜索扩展...**）。

## 已安装

每个扩展一张卡片，显示名称、版本、分类、描述和作者，加上：

- **启用/禁用开关** — 在不移除扩展的情况下启用或禁用它。
- **移除** — 再次点击**确认**以永久移除。

加载失败的扩展会在其卡片上显示错误信息。没有扩展时，标签会报告：“尚未安装任何扩展。”

## 浏览（目录）

**浏览**标签获取在线扩展目录（需要互联网连接）。每个目录条目都是一张带有**安装**按钮的卡片；已安装的扩展会显示**已安装**徽章。加载失败时会显示错误信息，并带有**重试**按钮。

## 从文件安装

- **ZIP** — 安装一个扩展 ZIP 文件（包含 `manifest.json` + `main.js`）。
- **JS** — 安装一个带有内嵌清单的单个 `.js` 文件。

安装后扩展会立即启用，任何功能区按钮也会立即出现。

## 通过扩展导入

**文件** → **导入**列出已安装扩展提供的导入格式；点击一种格式并选择文件。没有导入扩展时，该页面会报告：“未安装导入扩展。请通过“扩展”添加。”内置的导入格式（CSV、MS Project、P6）与此无关——参见 [Im-/export](docs://gids-import-export)。

## 编写您自己的扩展

扩展作者指南（清单、API、权限）位于代码仓库中：`github.com/OpenAEC-Foundation/open-planner-studio`，文件 `docs/extensions.md`。
