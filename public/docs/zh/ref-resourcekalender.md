# Resourcekalender

**资源日历**窗口编辑单个资源的自有日历——例如每周工作四天的班组。该表单与[日历对话框](docs://ref-kalenderdialoog)相同；本文只描述其中的差异。

## 打开方式

- 打开资源面板：**资源** → **管理**功能区组 → **资源**（完整面板）或**停靠**（停靠在甘特图旁边）。
- 在资源的**日历**列中，选择一个日历并点击旁边的铅笔图标（**编辑…**）进行编辑；通过同一个下拉菜单创建新日历。

## 与日历对话框的区别

- **一次一个日历** — 左侧没有库列表，也没有项目默认星标；只有表单。
- **应用**保存日历；**取消**、**Esc**、关闭图标或点击窗口外部会放弃更改。
- **没有自动重新计算** — 资源日历不会改变 CPM 日期；它计入负荷（直方图）和平衡，您需要分别通过 F5 或**平衡…**自行重新运行。

## 字段

完整字段参考请参见[日历对话框](docs://ref-kalenderdialoog)：**名称**、**工作日**（带有周一至周五和连续（24/7）预设）、**开始（时）** / **结束（时）** / **每日工时**、**工作时间**部分（小时计划开启时）、**生成假日…**和**假日**列表。
