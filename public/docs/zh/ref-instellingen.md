# Instellingen

**设置**窗口保存应用设置：适用于本设备、与项目文件无关的偏好设置。每次更改都会立即应用并保存——没有确定按钮。会改变已计算计划的排程选项则存储在项目中——参见 [Projectinformatie](docs://ref-projectgegevens)。

## 打开方式——三个入口，内容相同

- 标题栏中的**齿轮**（⚙）。
- **设置**（功能区标签）→ **项目**功能区组 → **设置**。
- **文件** → **设置**（Backstage）。

三者显示完全相同的设置，分布在三个标签中：**常规**、**语言**和**时间轴 / 缩放**。

## 常规标签

- **主题** — **深色**、**浅色**或**高对比度**；点击卡片切换。
- **文档切换样式** — 如何在打开的文档之间切换：**横向标签页**、**纵向标签页**或**胶囊**。
- **日期格式** — **dd-mm-yyyy**、**mm-dd-yyyy**或**yyyy-mm-dd**。仅影响显示；文件和计算不受影响。
- **版本** — 应用的版本号（只读）。
- **更新** — **检查更新**打开更新窗口。安装更新仅在桌面应用中可用；Snap 和 AppImage 安装通过各自的渠道更新。
- **默认缩放** — 默认缩放级别（只读，30 像素/天）。
- **调试终端** — **启用调试终端**显示用于排查问题的日志面板。
- **项目信息...** — 通往 [Projectinformatie](docs://ref-projectgegevens) 窗口的快捷方式。
- **导览** — **开始导览**重新播放入门导览。同一个重启入口也位于**视图**功能区标签 → **导览**，以及 Backstage（**文件** → **开始导览**）。

## 语言标签

- **语言** — 应用的显示语言；十四种语言，立即生效。

## 时间轴 / 缩放标签

- **小时计划** — **启用小时计划**开启小时/分钟排程：小时时间刻度、带工作时段的班次和精确到小时的任务条。关闭 ⇒ 应用保持完全按天粒度运行。开关开启后会出现**允许混合按天/按小时计划**（同一项目中的按天与按小时任务）。如果您打开一个包含小时计划、但开关处于关闭状态的文件，顶部会出现一条横幅，提供**启用小时计划**。参见 [Kalenders & uren-planning](docs://gids-kalenders-uren)。
- **工期显示** — **自动（按任务自身单位）**、**始终按天**或**始终按小时**。
- **中断处的任务条** — **从不拆分**、**选中时拆分**或**始终拆分**：任务条是否在非工作日周围视觉拆分。
- **一周起始日** — **星期一**或**星期日**（时间刻度的周布局）。
- **深度放大时显示刻钟** — 在小时时间刻度上的额外一刻钟刻度。
- **计算** — **自动计算**会在计划一旦过期时自动重新计算，而不等待按 F5。
- **滚动与缩放** — **模式**：
- **位置** — 由光标位置决定滚动方向；配合**屏幕分割**（**左/右**、**上/下**或**右上角**）。Ctrl+滚轮 = 缩放，Shift+滚轮 = 水平滚动。
- **按键** — 通过拖拽方块，为每种操作（**垂直**、**水平**、**缩放**）指定控制方式（**滚动**、**Ctrl + 滚轮**、**Shift + 滚轮**）；拖到已被占用的位置会交换两者的控制方式。
- **缩放 + 拖动** — 滚轮进行缩放（以光标为中心）；拖动图表背景以平移视图。
