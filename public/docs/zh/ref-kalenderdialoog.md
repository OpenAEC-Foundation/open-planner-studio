# Kalenderdialoog

**日历**窗口管理项目的日历库：左侧是所有日历的列表，右侧是所选日历的编辑表单。

## 打开方式

- **计划** → **日历**功能区组 → **日历**或**假日**按钮。
- **设置**（功能区标签）→ **日历**功能区组 → **日历**。
- 从项目向导：选择**自定义…**作为日历，会在创建后打开此窗口。

## 应用与取消

所有编辑——包括新建/复制/删除——都发生在一个工作副本中。**应用**（或 **Enter**）一次性写入所有内容并重新计算计划；**取消**、**Esc**、关闭图标或点击窗口外部会放弃所有更改。

## 库（左列）

- **列表** — 所有日历；星标标记**项目日历**（没有自己日历的任务所使用的默认日历）。
- **+** — **新建日历**。
- **复制** — 所选日历的副本。
- **删除** — 最后一个日历不能删除；删除项目默认日历会使另一个日历成为默认日历。
- **设为项目默认** — 将所选日历设为项目日历（表单上方的按钮）。

## 表单（右列）

- **名称** — 自由命名。
- **工作日** — **周一**至**周日**按钮；开 = 工作日。预设：**周一至周五**（标准工作周，07–16 时，8 小时/天）和**连续（24/7）**。
- **开始（时）** / **结束（时）** / **每日工时** — 全天的工作时间。一旦日历有了工作时段且小时计划开启，此项会隐藏；此后由工作时段驱动时间。

## 工作时间（仅在启用小时计划时可用）

- **推算的小时/天** — 由工作时段推算得出的核对数值。
- 预设：**白班**、**两班倒**、**三班倒**、**夜班**、**24/7** — 各自一次性设置好工作时段。
- **另存为预设…** — 将当前工作时间另存为您自己的预设（在此设备上）；自定义预设会显示为带删除图标的按钮。
- **按星期设置…** / **显示/隐藏工作时间** — 打开或收起时段编辑器。
- **时段编辑器** — 按星期几列出一份时段列表（开始–结束），每项都有一个**次日**复选框（跨越午夜的夜班）、**添加时段**（两个时段之间的间隔即为休息）、**复制到所有工作日**、每天的小时合计以及底部推算出的小时/天。参见 [Kalenders & uren-planning](docs://gids-kalenders-uren)。

## 生成假日…

按规则在项目周期内生成假日列表：

- **国家** — 荷兰、德国、比利时、法国、英国、奥地利、瑞士或**无假日**。
- **地区** — 仅适用于有地区分组的国家；默认**全国**。
- **建筑业假期** — 仅限荷兰：**无**、**北部**、**中部**或**南部**；带有这些是仅供参考日期的提示。
- **预览** — 摘要行（“n 个假日，年份–年份”），可展开为完整列表。
- **生成**会替换假日列表；**取消**关闭该区块。
- 如果项目现在延伸到已生成年份之外，顶部会出现提示，并带有**重新生成**按钮。

## 假日

列表本身：每行**描述**、**从**、**至**和一个移除按钮；**添加假日**创建新行。多天时段（建筑业假期、霜冻停工）只是一行跨度更长的从–至范围。
