# Baselinebeheer

**基线**窗口管理已保存的计划快照：保存、重命名、选择活动基线和删除。

## 打开方式

**计划** → **基线与进度**功能区组 → **保存基线…**或**管理基线…**（两者打开同一个窗口）。**Esc**、**关闭**、关闭图标或点击窗口外部会关闭窗口；此窗口中的所有更改会立即生效。

## 基线表格

每个已保存的基线一行：

- **活动** — 单选按钮；同一时间只能有一个基线是活动的。活动基线是甘特图叠加和差异报告的比较基准。
- **名称** — 可直接在行内编辑。
- **创建日期** — 保存基线的日期。
- **删除**（垃圾桶图标） — 移除该基线。若为活动基线，窗口会先要求确认（“删除当前基线吗？”）；之后最近保存的剩余基线会成为活动基线，若无剩余则无活动基线。

没有基线时，窗口显示“暂无基线”。

## 保存新基线

- **名称字段** — 预填为“基线 {n} — {date}”；可按需调整名称。
- **保存** — 记录每个任务的开始、完成日期（对里程碑而言是其日期），并将新基线设为活动。
- **警告** — 如果计划自上次计算以来已过期，会出现“计划已过期 — 请先重新计算（F5）”；这是一个提示，而非阻塞。针对过期计划保存的基线会冻结错误的日期。
